# Band Together

Band Together is a fullstack app that allows users to automatically generate an instant fan page for
their favorite bands.

User can save, remove and load their favorite artists to generate bio information, upcoming concert information
as well as a jukebox using API's from Spotify, Ticketmaster and MusicGraph.

User info and favorites persisted with Google Firebase.
